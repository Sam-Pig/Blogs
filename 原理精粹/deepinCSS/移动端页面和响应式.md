## 移动端页面和响应式

手机端页面的做法

1. 学会 media query
2. 学会要设计图（没图不做）     
实在要做也行，丑可别怪我
3. 学会隐藏元素
4. 手机端要加一个 meta  
    ```
    <meta name="viewport" content="width=device-width, user-scalable=no, initial-scale=1.0, maximum-scale=1.0, minimum-scale=1.0">
    ```
5. 手机端的交互方式不一样
    1. 没有 hover
    2. 有 touch 事件
    3. 没有 resize（浏览器窗口大小没办法改）
    4. 没有滚动条（隐藏的滚动条）
