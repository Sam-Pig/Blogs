## CSS的学习思路
1. 一、CSS 为什么难学（不正交 + 属性多 + 性价比低）
	> 一句话就是：“不正交”！
	1. position: fixed V.S. transform
	2. margin V.S. border
	3. 小圆点 V.S. display
	4. position: absolute V.S. display: inline
	5. css filter

2. 二、学不好 CSS 也没什么问题
	1. 记住几个常见的写法即可：两种水平居中、三种垂直居中、两种左右结构、两种左中右结构、一行省略、多行省略
	2. 别人研究出来我直接用即可（animate.css）

3. 三、想学好 CSS？
	1. 通读 specs
	2. 20篇以上博客（张鑫旭）
	3. 做好「学了但是没有什么卵用」的心理准备
